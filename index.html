Step 2 of 7

Install Vmware

Locate the VMware-Fusion .DMG file in your Downloads folder.

Double-click the .DMG file and follow the instructions.

During the installation:

  • accept all default settings
  • enter your Product Key when prompted

Once the installation has successfully completed, drag and drop the .DMG file into the Trash.

Step 4 of 7

3

Create a New Linux Virtual Machine

Launch VMware Fusion

  • Applications -> VMware Fusion
  1. File -> New...
  2. Click (notice: left-hand side with a coloured background): Install from disc or image
  3. Click: Continue
  4. Click: Use another disk or disc image...
  5. Browse your filesystem and find the Fedora ISO file you downloaded in the previous step.
  6. Click: Open
  7. Click: Continue
  8. Click: Customize Settings

Step 5 of 7

Customize VM Settings

  1. Save As (replace with): tuxserver
  2. Click: Save

System Settings -> General

Notes: Fedora Workstation (64-bit)

System Settings -> Processors & Memory

  • Processors: 4 processor cores (assuming you have 8 cores in total; select fewer cores if you don't have 8 total)
  • Memory: select the recommended value
  • Advanced Options:
    • Check: Enable hypervisor applications in this virtual machine
    • Check: Enable code profiling applications in this virtual machine

System Settings -> Display

Un-check: Accelerate 3D Graphics

Close the Settings window

  • click the red X located in the upper left hand side of the Settings window.

Step 6 of 7

Install Fedora Part1

  1. Click the white triangle to boot the Fedora operating system's Live installer.
  2. Wait for the Fedora operating system to complete its boot process.
  3. Click: Install to Hard Disk
  4. Select your preferred language during installation: English (Canada)
  5. Click: Continue
  6. Click: Localization -> Date & Time
    1. Region: Americas
    2. City: Toronto (or Montreal if you're a Habs fan)
    3. Click: Done
  7. Click: Network Configuration
    1. Replace the Hostname of localhost.localdomain with: tuxserver
    2. Click: Done
  8. Click: System -> Installation Destination
  9. Select: the VMware Disk
    1. Verify: 20 GiB
    2. Click: Done
  10. Click: Begin Installation (when it becomes available)

Step 7 of 7

Install Fedora Part2

  1. During the installation:
    1. Click: ROOT PASSWORD
      1. Set the Root Password to: abc123
      2. Confirm: abc123
      3. Click: Done
      4. Click: Done again to acknowledge root's password is weak.
    2. Click: User Creation
      1. Full name: your first name & last name
      2. Username (replace with): your username (e.g. bond0007)
      3. Check: Make this user administrator
      4. Set and confirm your password.
      5. Click: Done (to create your user account)
  2. Click: Finish configuration (when it becomes available)
  3. Click: Quit (when it becomes available)
  4. Reboot (aka restart):
    1. Click the power symbol in the upper right corner (look for the circle with a vertical line)
    2. Click the power symbol in the dialog.
    3. Click: Restart
  5. Wait for Fedora to restart.
